What is SecurityCam?


Web Development Project

As students from DevMountain, we came up with SecurityCam to use for our group project. SecurityCam was a way for us to showcase the skills we have learned while at DevMountain.


Security Camera System

We built a Security Camera using an Arduino Chip, webcam, and motion sensor. We also attatched a temprature sensor, a light meter, and a humidity sensor, so that we could display a wide range of data.


Single Page Web Application

The web application we made is used to display the photos taken from the security camera. There are many graphs that we also display to show the data from the other sensors as well.
